The VFPX Replacement Dialog

The VFPX version of the New Property/Method Dialog replacement adds a new Use Toggle Editor option to the dialog. For logical properties (those with a default value of .T. or .F.), turning this option installs a property editor that allows you to double-click the property in the Properties window to toggle its value. This property editor is defined in PropertyEditor.APP in the same directory as NewPropertyDialog.APP.

The VFPX version also fixes the following bugs:

  • It no longer gives an error if you use it to add a member to a subclassed ActiveX control.

  • Its caption now properly displays "Property" or "Method", depending on the setting of the Member Type option.

  • The maximum length of a property name is now 254 characters (formerly, it was 42).

  • The maximum length of the default value and description are now 252 characters (formerly, they were unlimited, and values longer than 254 characters caused an error).

To install the VFPX version, first install the VFP version as described in the New Property/Method Dialog Replacement topic. Then install the VFPX version somewhere on your system and DO NewPropertyDialog.APP to register it as the dialog to use.
