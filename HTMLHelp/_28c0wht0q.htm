Localizing the New Property/Method Replacement Dialog

Localizing the New Property/Method Replacement Dialog is easy:

  • Make a copy of NewPropertyDialogEnglish.H, renaming the copy to something else (for example, NewPropertyDialogGerman.H for a German language version).

  • Translate the strings in the copied H file.

  • Edit NewPropertyDialog.H and change the first line from:

    #include NewPropertyDialogEnglish.H

    to specify the name of the copied H file.
